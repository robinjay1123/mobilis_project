## рҹ‘Ҙ DETAILED ROLE WORKFLOWS

### 1пёҸвғЈ RENTER WORKFLOW
```
в”Ңв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”җ
в”Ӯ RENTER DASHBOARD (Newsfeed)        в”Ӯ
в”ңв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Ө
в”Ӯ 5 Tabs:                            в”Ӯ
в”Ӯ 1. HOME (Vehicle Discover)         в”Ӯ
в”Ӯ 2. BOOKINGS (My Rentals)           в”Ӯ
в”Ӯ 3. NOTIFICATIONS (Alerts)          в”Ӯ
в”Ӯ 4. MESSAGES (Chat)                 в”Ӯ
в”Ӯ 5. PROFILE (Settings)              в”Ӯ
в””в”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”ҳ

COMPLETE BOOKING LIFECYCLE:

1. BROWSE VEHICLES
   в”ңв”Җ Only is_available=true shown
   в”ңв”Җ Filter by category (Economy/SUV/Luxury/Van)
   в””в”Җ Sort by price, rating, popularity

2. SELECT VEHICLE & OPTIONS
   в”ңв”Җ Choose dates (start/end)
   в”ңв”Җ SELECT BOOKING TYPE:
   в”Ӯ  в”ңв”Җ Self-Drive (with_driver=false)
   в”Ӯ  в”Ӯ  "I will drive"
   в”Ӯ  в””в”Җ With Driver (with_driver=true)
   в”Ӯ     "I need a professional driver"
   в”ңв”Җ Pick location
   в””в”Җ Confirm booking

3. BOOKING CREATED
   в”ңв”Җ Status: PENDING
   в”ңв”Җ Payment: Process payment
   в””в”Җ Wait for operator approval

4. BOOKING AWAITING APPROVAL (OPERATOR'S DESK)
   в””в”Җ Renter can't do anything, just wait

5. BOOKING APPROVED вң…
   в”ңв”Җ Status: APPROVED
   в”ңв”Җ Notification sent
   в”ңв”Җ If with driver:
   в”Ӯ  в””в”Җ Driver assigned by operator
   в””в”Җ Ready for trip

6. TRIP STARTS
   в”ңв”Җ Status: ACTIVE
   в”ңв”Җ Driver (or partner) picks up renter
   в”ңв”Җ Real-time tracking (if with driver)
   в””в”Җ Try to reach destination

7. TRIP COMPLETES
   в”ңв”Җ Status: COMPLETED
   в”ңв”Җ Rate the vehicle (1-5 stars)
   в”ңв”Җ Rate the driver (if with driver service)
   в”ңв”Җ Leave comment
   в””в”Җ Payment processed

8. HISTORY
   в”ңв”Җ View past trips
   в”ңв”Җ See ratings given
   в””в”Җ Download receipt
```

### 2пёҸвғЈ PARTNER WORKFLOW
```
в”Ңв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”җ
в”Ӯ PARTNER DASHBOARD                  в”Ӯ
в”ңв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Ө
в”Ӯ 5 Tabs:                            в”Ӯ
в”Ӯ 1. DASHBOARD (Stats & Overview)    в”Ӯ
в”Ӯ 2. APPLICATIONS (Vehicle Status)   в”Ӯ
в”Ӯ 3. BOOKINGS (Rental Requests)      в”Ӯ
в”Ӯ 4. CONVERSATIONS (Chat)            в”Ӯ
в”Ӯ 5. NOTIFICATIONS (Alerts)          в”Ӯ
в””в”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”ҳ

WORKFLOW:

1. SIGN UP
   в””в”Җ role="partner"

2. OPTIONAL: GET VERIFIED
   в”ңв”Җ Upload ID
   в”ңв”Җ Face scan
   в””в”Җ Admin reviews

3. APPLY VEHICLE
   в”ңв”Җ Fill details:
   в”Ӯ  в”ңв”Җ Brand, model, year
   в”Ӯ  в”ңв”Җ Plate number
   в”Ӯ  в””в”Җ Price per day/hour
   в”ңв”Җ Upload documents:
   в”Ӯ  в”ңв”Җ OR (Official Receipt)
   в”Ӯ  в”ңв”Җ CR (Certificate of Registration)
   в”Ӯ  в””в”Җ Insurance docs
   в””в”Җ Submit for approval

4. VEHICLE APPLICATION STATUS
   в”ңв”Җ Applications tab shows status
   в””в”Җ Status: PENDING вҶ’ APPROVED/REJECTED

5. IF APPROVED вң…
   в”ңв”Җ Vehicle added to vehicles table
   в”ңв”Җ Operator can now toggle is_available
   в”ңв”Җ When is_available=true:
   в”Ӯ  в””в”Җ Shows in renter's newsfeed
   в””в”Җ Bookings start coming in

6. RECEIVE BOOKINGS
   в”ңв”Җ Bookings tab shows requests
   в”ңв”Җ Shows:
   в”Ӯ  в”ңв”Җ Renter info
   в”Ӯ  в”ңв”Җ Dates & duration
   в”Ӯ  в”ңв”Җ Booking type:
   в”Ӯ  в”Ӯ  в”ңв”Җ Self-drive (partner not involved)
   в”Ӯ  в”Ӯ  в””в”Җ With driver (partner might drive)
   в”Ӯ  в””в”Җ Payment amount
   в””в”Җ STATUS: PENDING (operator must approve)

7. WAIT FOR OPERATOR APPROVAL
   в”ңв”Җ Partner can't approve/reject
   в”ңв”Җ Operator decides in their panel
   в””в”Җ Two scenarios:
      в”ңв”Җ IF Partner is_driver=true + with_driver=true:
      в”Ӯ  в””в”Җ Partner drives, no driver assignment needed
      в””в”Җ IF Partner is_driver=false OR Self-drive:
         в””в”Җ Operator assigns company driver (or none for self-drive)

8. IF BOOKING APPROVED вң…
   в”ңв”Җ Status changes to APPROVED
   в”ңв”Җ If partner drives:
   в”Ӯ  в”ңв”Җ Partner gets pickup notification
   в”Ӯ  в”ңв”Җ Drives renter to destination
   в”Ӯ  в””в”Җ Gets payment when completed
   в””в”Җ If self-drive:
      в””в”Җ Renter drives (partner doesn't participate)

9. EARNINGS
   в”ңв”Җ Earnings tracked after trip completion
   в”ңв”Җ Payment automatically processed
   в””в”Җ Partner sees earnings in dashboard

10. RATINGS
    в””в”Җ View ratings from renters
```

WORKFLOW:

   в”ңв”Җ role="driver"
   в”ңв”Җ Basic info (name, email, phone, address)
   в””в”Җ Account created

2. OPTIONAL: GET VERIFIED
   в”ңв”Җ Upload License:
   в”Ӯ  в”ңв”Җ License number
   в”Ӯ  в”ңв”Җ Expiry date
   в”Ӯ  в”ңв”Җ Photo (front)
   в”Ӯ  в””в”Җ Photo (back)
   в”ңв”Җ Upload NBI Clearance:
   в”Ӯ  в”ңв”Җ Clearance number
   в”Ӯ  в”ңв”Җ Expiry date
   в”Ӯ  в””в”Җ Document file
   в””в”Җ Set availability:
      в”ңв”Җ Preferred work days
      в”ңв”Җ Preferred areas
      в””в”Җ Work hours

   в”Ӯ  в”ңв”Җ License valid & not expired?
   в”Ӯ  в”ңв”Җ NBI clearance valid & not expired?
   в”Ӯ  в””в”Җ Background check passed?

   в”Ӯ
   в”ңв”Җ IF APPROVED вң…:
   в”Ӯ  в”ңв”Җ drivers.verification_status = "approved"
   в”Ӯ  в”ңв”Җ Assign tier: Standard (default) / Professional / Elite
   в”Ӯ  в”ңв”Җ user.is_available = true
   в”Ӯ  в””в”Җ Driver can now receive job offers
   в”Ӯ
   в””в”Җ IF REJECTED вқҢ:
      в”ңв”Җ drivers.verification_status = "rejected"
      в”ңв”Җ Reason provided
      в””в”Җ Can reapply after fixing issues

   в”ңв”Җ Availability tab
   в”ңв”Җ Toggle: AVAILABLE / UNAVAILABLE
   в”ңв”Җ Set work schedule:
   в”Ӯ  в”ңв”Җ Select days: Mon, Tue, Wed, etc.
   в”Ӯ  в”ңв”Җ Set hours: 8AM - 6PM (example)
   в”Ӯ  в””в”Җ Preferred areas
   в””в”Җ Only receives offers when AVAILABLE

   в”ңв”Җ Operator assigns booking to driver
   в”ңв”Җ Job offer notification (push notification)
   в”ңв”Җ Shows in JOBS tab:
   в”Ӯ  в”ңв”Җ Renter name
   в”Ӯ  в”ңв”Җ Pickup location + time
   в”Ӯ  в”ңв”Җ Dropoff location
   в”Ӯ  в”ңв”Җ Estimated earnings
   в”Ӯ  в””в”Җ 2 buttons: ACCEPT / DECLINE
   в””в”Җ Driver has 5 minutes to respond (or auto-decline)

   в”ңв”Җ driver_job_assignments.status = "accepted"
   в”ңв”Җ Status changes to ACTIVE
   в”ңв”Җ Driver gets:
   в”Ӯ  в”ңв”Җ Renter phone number
   в”Ӯ  в”ңв”Җ Detailed route
   в”Ӯ  в””в”Җ Vehicle to use (if company car)
   в”ңв”Җ Notification sent to:
   в”Ӯ  в”ңв”Җ Renter: "Your driver is coming"
   в”Ӯ  в””в”Җ Operator: "Job confirmed"
   в””в”Җ Trip starts

7. ACTIVE TRIP
   в”ңв”Җ Driver sees:
   в”Ӯ  в”ңв”Җ Real-time GPS navigation
   в”Ӯ  в”ңв”Җ Renter's exact location
   в”Ӯ  в”ңв”Җ Pickup location
   в”Ӯ  в”ңв”Җ Dropoff: location
   в”Ӯ  в””в”Җ Timer showing elapsed time
   в”ңв”Җ Driver navigation:
   в”Ӯ  в”ңв”Җ Navigate to renter (pickup)
   в”Ӯ  в”ңв”Җ Pick up renter
   в”Ӯ  в”ңв”Җ Navigate to destination (dropoff)
   в”Ӯ  в””в”Җ Confirm arrival
   в””в”Җ Renter sees driver's real-time location

8. TRIP COMPLETION
   в”ңв”Җ Driver marks trip COMPLETE
   в”ңв”Җ Renter confirms dropoff
   в”ңв”Җ driver_trips entry created:
   в”Ӯ  в”ңв”Җ Distance traveled
   в”Ӯ  в”ңв”Җ Duration
   в”Ӯ  в””в”Җ Final payment
   в”ңв”Җ Renter rates driver (1-5 stars)
   в”ңв”Җ Earnings calculated:
   в”Ӯ  в”ңв”Җ Trip fee
   в”Ӯ  в”ңв”Җ Minus PSDC commission (15% or custom)
   в”Ӯ  в””в”Җ = Net earnings for driver
   в””в”Җ Payment queued for payout

9. EARNINGS HISTORY
   в”ңв”Җ EARNINGS tab shows:
   в”Ӯ  в”ңв”Җ Today's earnings
   в”Ӯ  в”ңв”Җ This week's earnings
   в”Ӯ  в”ңв”Җ This month's earnings
   в”Ӯ  в”ңв”Җ Breakdown:

       в”ңв”Җ Vehicle booked

       в””в”Җ Operator reviews

   рҹ“Ң SCENARIO A: SELF-DRIVE BOOKING

   в”ңв”Җ Renter will drive themselves

   в”ңв”Җ Operator clicks: "Approve Booking"

   в”Ӯ  в””в”Җ Company (if company vehicle)

   рҹ“Ң SCENARIO B: WITH DRIVER + PARTNER DRIVES
   в”ңв”Җ with_driver = true
   в”ңв”Җ Partner is_driver = true
   в”ңв”Җ Partner will drive their own vehicle
   в”ңв”Җ Operator action: Just approve вң…
   в”ңв”Җ Operator clicks: "Approve Booking"
   в”ңв”Җ Status вҶ’ APPROVED
   в”ңв”Җ Booking fee goes to:
   в”Ӯ  в”ңв”Җ Partner (as vehicle owner + driver)
   в”Ӯ  в””в”Җ PSDC takes commission
   в””в”Җ Partner gets pickup notification

   рҹ“Ң SCENARIO C: WITH DRIVER + NEEDS ASSIGNMENT

   в”ңв”Җ Partner is_driver = false OR Company vehicle

   в”Ӯ  в”ңв”Җ Within preferred areas
   в”Ӯ  в””в”Җ Within preferred hours
   в”ңв”Җ Operator selects a driver from list
   в”ңв”Җ Job offer sent to driver (driver gets notification)
   в”ңв”Җ Booking status вҶ’ APPROVED
   в”ңв”Җ Booking fee split:
   в”Ӯ  в”ңв”Җ Driver fee: 60-70% (configurable)
   в”Ӯ  в”ңв”Җ Partner/Company: 20-30%
   в”Ӯ  в””в”Җ PSDC commission: rest
   в””в”Җ Driver has 5 mins to accept/decline

   в”ңв”Җ If driver declines job
   в”ңв”Җ Offer goes back to operator
   в”ңв”Җ Operator can:
   в”Ӯ  в”ңв”Җ Offer to another driver
